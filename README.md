# Atividades com VETORES 💻

## 🚀 Tecnologias Utilizadas

* **Linguagem:** JavaScript (Node.js)
* **Ferramenta de Edição:** Visual Studio Code
* **Controle de Versão:** Git e GitHub

Este repositório contém uma lista de exercícios desenvolvidos para praticar a manipulação de estruturas de vetores (arrays), laços de repetição e condicionais em JavaScript.

---

## 🎯 Resultados Esperados

* Programas feitos em JavaScript (extensão `.js`) que representem a resolução dos problemas propostos.
* Programas funcionando corretamente após testados.
* Estruturas de vetores utilizadas para aprimorar a lógica dos algoritmos.
* Versionamento no repositório local e remoto de todos os arquivos dos algoritmos propostos.

---

## 📝 Descrição dos Exercícios

### 1. Maior Idade e Posição
Crie um programa que leia a idade de 4 pessoas e guarde-as em um vetor. No final, mostre qual foi a maior idade digitada e em qual posição (índice) do vetor ela foi armazenada.

### 2. Filtro de Funcionárias (Salário > R$ 5 mil)
Faça um programa que leia o nome, o sexo e o salário de 5 funcionários e guarde esses dados em três vetores diferentes. No final, mostre uma listagem contendo apenas os dados das funcionárias mulheres que ganham mais de R$ 5.000,00.

### 3. Análise de Alturas Dinâmica
Desenvolva um programa que colete e armazene os nomes e as alturas de diferentes pessoas em vetores. O programa deve permitir que o usuário insira quantos dados desejar e só deve ser encerrado quando o usuário optar por parar. Ao final, exiba a maior altura, a menor altura e a quem elas pertencem respectivamente.
